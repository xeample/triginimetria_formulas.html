Основное тригонометрическое тождество

sin²α + cos²α = 1;

tg²α + 1 = 1 / cos²α;

1 + ctg²α = 1 / sin²α;

Формулы сложения

sin(α + β) = sin α cos β + cos α sin β;

sin(α − β) = sin α cos β − cos α sin β;

cos(α + β) = cos α cos β − sin α sin β;

cos(α − β) = cos α cos β + sin α sin β;

tg(α + β) = (tg α + tg β) / (1 − tg α tg β);

tg(α − β) = (tg α − tg β) / (1 + tg α tg β);

ctg(α + β) = (ctg α ctg β − 1) / (ctg α + ctg β);

ctg(α − β) = (ctg α ctg β + 1) / (ctg β − ctg α);

Формулы приведения

с опорной точкой π/2:

sin(π/2 − α) = cos α;

cos(π/2 − α) = sin α;

tg(π/2 − α) = ctg α;

ctg(π/2 − α) = tg α;

sin(π/2 + α) = cos α;

cos(π/2 + α) = −sin α;

tg(π/2 + α) = −ctg α;

ctg(π/2 + α) = −tg α;

с опорной точкой π:

sin(π − α) = sin α;

cos(π − α) = −cos α;

tg(π − α) = −tg α;

ctg(π − α) = −ctg α;

sin(π + α) = −sin α;

cos(π + α) = −cos α;

tg(π + α) = tg α;

ctg(π + α) = ctg α;

с опорной точкой 3π/2:

sin(3π/2 − α) = −cos α;

cos(3π/2 − α) = −sin α;

tg(3π/2 − α) = ctg α;

ctg(3π/2 − α) = tg α;

sin(3π/2 + α) = −cos α;

cos(3π/2 + α) = sin α;

tg(3π/2 + α) = −ctg α;

ctg(3π/2 + α) = −tg α;

с опорной точкой 2π:

sin(2π − α) = −sin α;

cos(2π − α) = cos α;

tg(2π − α) = −tg α;

ctg(2π − α) = −ctg α;

sin(2π + α) = sin α;

cos(2π + α) = cos α;

tg(2π + α) = tg α;

ctg(2π + α) = ctg α;

Формулы двойного угла

sin 2α = 2 sin α cos α;

cos 2α = cos²α − sin²α;

cos 2α = 2 cos²α − 1;

cos 2α = 1 − 2 sin²α;

tg 2α = (2 tg α) / (1 − tg²α);

ctg 2α = (ctg²α − 1) / (2 ctg α);

Формулы тройного угла

sin 3α = 3 sin α − 4 sin³α;

cos 3α = 4 cos³α − 3 cos α;

tg 3α = (3 tg α − tg³α) / (1 − 3 tg²α);

Формулы понижения степени

cos²α = (1 + cos 2α) / 2;

sin²α = (1 − cos 2α) / 2;

Формулы тангенса половинного угла

tg²(α/2) = (1 − cos α) / (1 + cos α);

tg(α/2) = (1 − cos α) / sin α;

tg(α/2) = sin α / (1 + cos α);

Универсальная подстановка

sin α = (2 tg(α/2)) / (1 + tg²(α/2));

cos α = (1 − tg²(α/2)) / (1 + tg²(α/2));

tg α = (2 tg(α/2)) / (1 − tg²(α/2));

ctg α = (1 − tg²(α/2)) / (2 tg(α/2));

Суммы и произведения тригонометрических функций

sin α + sin β = 2 sin((α + β)/2) cos((α − β)/2);

sin α − sin β = 2 sin((α − β)/2) cos((α + β)/2);

cos α + cos β = 2 cos((α + β)/2) cos((α − β)/2);

cos α − cos β = 2 sin((α + β)/2) sin((β − α)/2);

tg α + tg β = sin(α + β) / (cos α cos β);

tg α − tg β = sin(α − β) / (cos α cos β);

ctg α + ctg β = sin(α + β) / (sin α sin β);

ctg α − ctg β = sin(β − α) / (sin α sin β);

tg α + ctg β = cos(α − β) / (cos α sin β);

ctg α − tg β = cos(α + β) / (sin α cos β);

2 cos α cos β = cos(α + β) + cos(α − β);

2 sin α sin β = cos(α − β) − cos(α + β);

2 sin α cos β = sin(α + β) + sin(α − β);
